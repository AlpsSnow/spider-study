# CB_spider
通过爬虫抓取[集思录](https://www.jisilu.cn/data/cbnew/#cb)网站的可转债信息，分析当前可转债市场的投资价值。

## 安装
首先，安装Python3，并配置好环境。

使用 pip 安装依赖:

```
pip3 install -r require.txt

```

## 运行：
CB_spider.py

## 示例：
```
D:\python work\spider-study\CB_spider>CB_spider.py
```
